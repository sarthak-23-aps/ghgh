What is JavaScript? +

JavaScript is a scripting language used to create and control dynamic website content, such as interactive forms, animations, and more.

What is HTML used for? +

HTML stands for HyperText Markup Language. It's used to structure web content like headings, paragraphs, links, and images on websites.

What is CSS? +

CSS (Cascading Style Sheets) is used to style HTML elements. It controls layout, colors, fonts, and overall look of the webpage.

What is DOM in JavaScript? +

DOM stands for Document Object Model. It's a programming interface that allows JavaScript to manipulate HTML and CSS on the page.

What is a variable in programming? +

A variable is a container used to store data values. In JavaScript, you can declare variables using let, const, or var.

What is an array? +

An array is a special variable that can hold multiple values at once. Each value has an index starting from 0.
